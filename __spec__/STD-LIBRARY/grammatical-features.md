# Grammatical Features

Armenian has seven noun cases

Case | Description | Example
Nominative | Subject of the sentance | goshik
Accusative | Direct Object | goshik
Genitive | Possessive | goshik**i**
Dative | Indirect Object | goshik**i**
Ablative | From | goshik**its**
Instrumental | Using something | goshik**ov**
Locative | In something | goshik**um**
