# Variables and Types

## Declaring a variable
Variables are declared with the `նոր` or `nor` (new) keyword, followed by the type and the variable name in the definitive form.

```
նոր տող բանը «Մեկենա» է։
նոր թիվ երկերորդ-բանը 2 է։

nor togh ban@ "Mekena" e.
nor tiv erkerord_ban@ 2 e.
```
